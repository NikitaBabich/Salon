# Заголовок проекта

Проект по учету клиентов и посещения ими услуг в салоне красоты "Мордочка".

## Начало работы

Эти инструкции предоставят вам копию проекта и помогут запустить на вашем локальном компьютере для разработки и тестирования.

### Необходимые условия

Для установки программы необходим архиватор, любая среда разработки с поддрежкой языка C#. Для устаноывки прогрвммы необходимо извлечь фалйы программы в люую доступную папку

### Установка

Для работы с программой необходимо извлечь файлы программы в любую папку, затем запустить файл Salon.sln в среде пррограммирования для изменения программы. 

Для работы с программой без ее изменения необходимо звпустить файл Salon.exe, который находить в Salon\bin\Debug
